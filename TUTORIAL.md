# TermKit Tutorial

## Quick Start

Run these in order:

```bash
chmod +x ghostty-setup-catppuccin.sh terminal-aesthetic-setup.sh
./ghostty-setup-catppuccin.sh
./terminal-aesthetic-setup.sh
source ~/.zshrc
```

## Most Used Commands

```bash
ll          # List files with details
lt          # Tree view
cat file    # View file with colors
z dir       # Jump to directory
grep text   # Search in files
sysinfo     # System information
```

For complete documentation, see the full TUTORIAL.md in the repo.
